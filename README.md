# veeve


  ## About veeve
 **veeve** is a console program that allows the user to do perform the following actions
 
 * download youtube videos 
 * download youtube playlists
  
  **Note:** The prgogram only allows the user to download videos off of youtube 
 
 **veeve** displays the available resolutions for the video and allows the user to download the video at their desired resoltion
